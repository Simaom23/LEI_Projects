Ex.2 - Utilizar as diferentes flags -E -S -c explicando o que ocorre em cada uma:
		-E - Ocorre print das instruções de pré-processamento de sum_v1.
		-S - É criado um ficheiro sum_v1.s que contém o código em linguagem Assembly do programa sum_v1.
		-c - É criado um ficheiro sum_v1.o que contém o código máquina do programa sum_v1.
